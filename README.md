# 人工智能数学原理与算法B（AIB）

课程实验记录，涵盖机器学习基础、深度学习基础、自然语言处理和 AI for Science 等专题。

## 知识框架

- **机器学习基础**：数据预处理、回归、分类、聚类、降维、模型评估
- **深度学习基础**：全连接网络、卷积神经网络、循环神经网络
- **自然语言处理**：基于 RNN / Transformer 的情感分析
- **AI for Science**：文献调研与综述

## 代码与数据来源

- 实验代码模板及数据集均来自课程群发布内容
- 代码中若干部分需要自行补全实现

## 环境配置

个人使用 WSL + Miniconda + Python 3.12+，依赖通过 pip 安装。

也可使用学校提供的算力平台。也可强行在自己的 Windows 上跑。
