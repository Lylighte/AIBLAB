## 1. 背景与问题 (1 min)

- **什么是符号回归？** — 从数据中自动挖掘数学表达式，不是黑箱预测
- **研究问题：** 纳米粒子 OR 熟化中的生长级数方程
- **目标方程：** $$q = a \frac{\alpha^{1/3} \gamma \Omega}{R \cdot kT} + b$$
- **算法基础：** SISSO = 特征扩展 + 独立筛选 + 系数拟合

## 2. 实验 7.6.2 — 两种求解方法的比较 (2 min)

### 核心问题
同样的线性回归，解析解和神经网络解分别适合什么场景？

### 算法对比

| | 解析解 | 神经网络 |
|---|---|---|
| **公式** | $$c = (X^TX)^{-1}X^Ty$$ | 梯度下降 min MSE |
| **计算** | 矩阵求逆 $$O(d^3)$$ | $$O(n \cdot d \cdot epoch)$$ |
| **优势** | 一步到位，精确解 | 避免求逆，可扩展 |
| **劣势** | d 大时不可行 | 需调参，迭代慢 |

### 关键结果（填入实际数据）
- SIS 开启 (d=10): 解析解 _ ms vs NN _ ms → 解析解占优
- SIS 关闭 (d=____): 解析解 _ s vs NN _ s → NN 开始有优势
- **结论:** 小空间解析解，大空间 NN，两者互补

## 3. 实验 7.6.3 — 重构特征扩展 (3 min)

### 基础框架的局限性
现有 expansion 只有 $$x^p$$ + $$a \times b$$，无法产生目标方程需要的：
- ❌ 除法: $$\frac{\alpha^{1/3} \gamma \Omega}{R \cdot kT}$$
- ❌ 三元组合: $$\alpha^{1/3} \times \gamma \times \Omega$$

### 解决方案: 逐步扩充运算
新增 4 种运算函数: `combine_divide`, `combine_3`, `combine_3_mixed`, `log_transform`, `exp_transform`

### 空间膨胀分析（核心数据）

| 步骤 | 新增运算 | 特征数 | 增长倍率 | 含目标? |
|---|---|---|---|---|
| Baseline | 幂次+二元乘 | ___ | 1× | ❌ |
| +二元除 | a/b, b/a | ___ | ___× | ❌ |
| +三元乘 | a×b×c | ___ | ___× | ✅ |
| +三元混合 | a×b/c 等 | ___ | ___× | ✅ |

### 发现
- 除法的引入是关键（目标本质是分式）
- 三元组合导致组合爆炸（C(30,3) ≈ 4000+ 基础组合）
- 盲目全空间搜索代价极高 → 引出 7.6.4

## 4. 实验 7.6.4 — 迭代式针对性搜索 ⭐ (4-5 min)

### 设计动机
7.6.3 的问题：一次性盲目扩充 → 99% 的特征是无意义的 → 浪费计算且不可解释

### 四种情况
**从"盲搜"变为"聚焦"**：用上轮好结果指导下轮

### 算法框架 (四模块)

```
特征扩展 → SIS筛选 → 系数拟合 → top-K 种子
    ↑                              ↓
    └──── 导向性扩展 ←──────────────┘
                    ↓
               终止判断 → 输出最优公式
```

### 模块 1: 导向性扩展
- **三种策略:** 替换 / 交叉 / 分层 (推荐)
- **分层策略:** 保留原始物理特征 + 高分种子交叉组合
- **去重:** 用 `seen_names` 集合避免重复探索

### 模块 2: 树状结果存储
- 追踪公式的推导路径: 原始特征 → 中间特征 → ... → 最终公式
- 可解释性的关键 — 不是黑箱输出，而是有逻辑的推导过程

### 模块 3: 终止策略
- 目标达成: R² > 0.999
- 滑动窗口收敛: 最近 5 轮无提升
- 最大迭代: 防止无限循环

## 5. 总结 (1 min)

### 三阶段递进逻辑
```
基础框架 → 发现无法学到目标 (缺除法/三元)
    ↓
全空间搜索 → 能学到但代价巨大 (组合爆炸)
    ↓
迭代搜索 → 高效 + 准确 + 可解释
```

### 思考题要点
1. **SIS 筛选策略:** 逐维贪心 vs 多维联合筛选
2. **可解释性增强:** 量纲分析 + 物理约束 + 推导树追溯
3. **AI 融合:** LLM 做假设生成 + 符号回归做验证 + 物理知识库做过滤
